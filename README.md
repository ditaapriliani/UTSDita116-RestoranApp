# Eleven Eatery Mobile App

## Deskripsi Aplikasi
Eleven Eatery adalah aplikasi mobile bertema restoran modern yang dirancang untuk memudahkan pengguna dalam melihat daftar menu makanan dan minuman, mencari menu favorit, melihat detail menu, serta mengetahui lokasi restoran dengan tampilan antarmuka yang modern dan nyaman digunakan.

Aplikasi ini memiliki desain minimalis dengan dominasi warna merah dan cream yang memberikan kesan hangat serta elegan.
